# Hackaton project boilerplate

## Requirements

- php 7.4
- Composer 2
- node.js 14
- npm 6
- git

## Installation

	git clone git@github.com:nessai1/hackathon.git
	cd hackathon
	composer install
	cp .env.example .env
	php artisan key:generate
	npm install
	npm run watch
	php artisan serve

Now project works on [http://localhost:8000/](http://localhost:8000/)
